## 📦 Installation

### Option 1 — Clone and copy
\`\`\`bash
git clone https://github.com/luokai0/cloud-skills-by-luo-kai-.git
cp -r cloud-skills-by-luo-kai-/skills/* .agents/skills/
\`\`\`

### Option 2 — Git submodule
\`\`\`bash
git submodule add https://github.com/luokai0/cloud-skills-by-luo-kai-.git .agents/codingskills
\`\`\`

## 🛠️ Usage

Once installed, just ask your AI agent anything in that domain — it will automatically apply the right skill.
