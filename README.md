## 🚀 How to Use

Each skill is a `SKILL.md` file following the standard Agent Skills format. Place any skill file into your AI coding agent's skills directory:

**Claude Code:**
```bash
# Copy to your project's .claude/skills/ folder
cp skills/rust-expert/SKILL.md .claude/skills/rust-expert.md
```

**Cursor / Windsurf / Codex:**
```bash
# Most support loading from .cursor/rules/ or similar
cp skills/python-expert/SKILL.md .cursor/rules/python-expert.md
```

**Claude.ai (claude.ai UI):**
Skills in `/mnt/skills/` are loaded automatically when the platform supports it.

## ✍️ Skill Format

Every skill follows this standard YAML + Markdown format:

```yaml
---
author: luo-kai
name: skill-name
description: When to use this skill. Trigger phrases and conditions.
license: MIT
metadata:
  author: luokai0
  version: "1.0"
  category: coding
---

# Skill Title

## Before Starting
Questions to ask before diving in.

## Core Expertise Areas
Key domains this skill covers.

## Key Patterns & Code
Real code examples with explanations.

## Best Practices
Production-tested guidelines.

## Common Pitfalls
| Pitfall | Symptom | Fix |
```
